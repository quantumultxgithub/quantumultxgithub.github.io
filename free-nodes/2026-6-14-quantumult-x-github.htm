设置代理模式与规则

配置完节点后，下一步就是设置代理规则。Quantumult提供三种代理模式：

1. 全局代理模式

所有流量均通过代理节点转发。适合访问国外内容或绕过全部网络限制。

2. 按需代理模式（策略组）

通过自定义规则决定哪些域名或IP使用代理，其余直连。适合日常混合使用场景。

3. 手动规则模式

通过导入规则列表进行精细化管理。高级用户可用自定义规则控制不同流量走向。

推荐：使用“策略组 + 常用规则列表（如ACL4SSR）”组合，兼顾效率与自由。

常见问题解答（FAQ）

Q1：Quantumult支持哪些协议？

A：目前支持的协议包括Shadowsocks（SS）、ShadowsocksR（SSR）、Vmess、Trojan等，足以应对大部分使用场景。

Q2：Vmess相比其他协议有什么优势？

A：Vmess拥有更强的加密机制与抗干扰能力，支持动态端口、WebSocket、TLS伪装等高级功能，隐蔽性更强。

Q3：为什么节点无法连接？

A：可能原因包括：节点地址失效、UUID错误、端口阻断、网络问题、TLS证书配置错误等。建议逐项排查。

Q4：Quantumult是否支持多平台？

A：Quantumult仅限iOS使用，但配置可通过iCloud导出/导入，实现多台iOS设备间同步。

Q5：Quantumult与Quantumult X有何不同？

A：Quantumult为早期版本，功能稳定、界面简洁；Quantumult X为后续增强版，增加了更多高级功能如JS脚本等。两者互不兼容，需按需选择。

FAQ

VMess 使用 AEAD 加密后连接性能如何？
AEAD 加密在保证数据安全的同时，性能开销较低，抗重放攻击能力强。高封锁环境下更稳定，同时减少流量特征泄露，适合长时间和高安全性需求的网络连接。
如何给 Clash 添加自定义域名白名单以提升访问体验？
在 rules 中添加 `DOMAIN-SUFFIX,example.com,DIRECT` 或 `DOMAIN-KEYWORD,example,DIRECT` 条目，将常用且可靠的站点设为直连，可减少代理绕行延迟并避免因代理造成的登录/验证问题。
Trojan-go 如何启用 WebSocket 支持？
在配置文件中将 transport 参数设置为 websocket，并指定 path 路径。客户端需同步使用相同路径及主机名。这样可将 Trojan 流量伪装为普通 WebSocket 通信以提升隐蔽性。
Sing-box 的规则订阅更新机制是怎样的？
Sing-box 支持远程规则订阅 URL，可自动下载并更新节点和分流规则。用户可设置更新频率，如每天或每小时，确保规则和节点始终保持最新状态，提高网络管理效率。
Mihomo 的 Rule-Set 与 Clash Premium 是否兼容？
Mihomo 完全兼容 Clash Premium 的 Rule-Set 格式，支持相同的远程规则订阅与策略分组。用户可直接迁移原 Clash 配置，无需额外修改或转换。
WinXray 的多出口节点如何优化访问速度？
通过为不同流量类型设置不同出口节点，WinXray 可以自动选择延迟最低的节点处理请求。结合策略组，可提高网页访问、视频播放和游戏等应用的速度与稳定性。
V2Ray 的 gRPC 模式适合哪些场景？
gRPC 支持多路复用和长连接，适合高并发访问和移动网络场景。相比 TCP 或 WebSocket，它可以减少握手次数，降低延迟，同时提高连接稳定性和吞吐量，特别适合游戏或即时通讯。
Trojan 伪装域名如何选择以提升隐蔽性？
伪装域名应选择真实可访问网站并配备有效证书，这样 Trojan 流量可模拟正常 HTTPS 请求，降低被封锁和检测的风险，提高节点长期可用性和访问稳定性。
Sing-box 如何按 IP 段实现智能分流？
Sing-box 支持根据 IP 段匹配流量，将国内流量直连、国外流量走代理。结合策略组和规则文件，实现精细化流量管理，保证访问速度和稳定性，同时提高节点使用效率。
QuantumultX 支持哪些类型的重写脚本？
QuantumultX 支持 JavaScript 脚本，可修改请求头、响应内容、自动跳转、广告屏蔽和定制流量处理。结合策略组，可实现高度个性化的网络访问控制和优化。
